# Quiz Master Pro - Улучшенная онлайн викторина

## 🎮 Особенности игры

### Новые возможности:
1. **Настройка количества вопросов** - от 5 до 30 вопросов
2. **Три уровня сложности**:
   - Легкий (простые вопросы, 4-5 вариантов ответа)
   - Средний (более сложные вопросы, 4-6 вариантов ответа)
   - Сложный (очень сложные вопросы, 5-7 вариантов ответа)
3. **Настройка времени** - от 5 до 60 секунд на вопрос
4. **Система ошибок** - ограничение ошибок на игрока (0-5 ошибок)
5. **Бонусы за скорость** - чем быстрее ответ, тем больше очков
6. **Разное количество вариантов ответов** - от 4 до 7 в зависимости от сложности
7. **Улучшенный дизайн** - современный glassmorphism эффект


## 🚀 Установка и запуск

### 1. Установка сервера
# Перейдите в папку server, в строке где указан путь напишите:
Cmd

# Установите на оффициальном сайте Node.js

# В командной строке пропишите и установите зависимости
npm install

# Запустите сервер
npm start

# Перейдите по хосту
http://localhost:3000
